[h2]Overview[/h2] [img]https://i.imgur.com/jdunrUl.png[/img]
This mod is a collection of small, vanilla-consistent tweaks suited to any type of playthrough. This mod is 100% configurable, so ANY feature can be toggled based on your preferences.

[h2]Compatibility[/h2] [img]https://i.imgur.com/jdunrUl.png[/img]
This mod adds a few new building blocks used by [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2861542679][b]Remnants[/b][/url] in its world generation, so it naturally requires this mod to also be installed.
